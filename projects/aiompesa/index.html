aiompesa

Mon Jul 22, 2019

Tags: mpesa | asyncio | python |

Motivation

The reason I wrote this library is because:

  1. I wanted to learn some more about asyncio.
  2. To create an asynchronous library.
  3. Experiment with Python 3+ static typing.

Results

I understand a little more about asyncio and the various IO problems that it is trying to solve. Maybe you might not need to make async M-Pesa calls now in your product but, it’s something you can consider in future. The aiompesa library is available on pypy and open sourced at GitHub.
